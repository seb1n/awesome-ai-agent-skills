# SEO Optimization

This skill enables the agent to optimize a webpage for search engines. The agent can analyze keywords, improve content, and suggest technical SEO improvements to increase the visibility of the webpage in search results.

## Workflow

1.  **Keyword analysis:** The agent will research and identify relevant keywords for the webpage.
2.  **Content optimization:** The agent will analyze the content of the webpage and suggest improvements to make it more relevant and engaging for the target audience.
3.  **On-page SEO:** The agent will check for and suggest improvements to on-page SEO factors, such as title tags, meta descriptions, and header tags.
4.  **Technical SEO:** The agent will check for and suggest improvements to technical SEO factors, such as page speed, mobile-friendliness, and structured data.
5.  **Generate an SEO report:** The agent will produce a report that summarizes the findings and provides a prioritized list of recommendations for improvement.

## Usage

To use this skill, provide the agent with the URL of the webpage you want to optimize.

**Example:**

```
Optimize the following webpage for the keyword "AI agent skills":

https://example.com/ai-agent-skills
```
