# Project Management

This skill enables the agent to manage a software project. The agent can create and assign tasks, track progress, and facilitate communication among team members.

## Workflow

1.  **Create a project plan:** The agent will work with the user to create a project plan, including a list of tasks, deadlines, and dependencies.
2.  **Assign tasks to team members:** The agent will assign tasks to team members based on their skills and availability.
3.  **Track progress:** The agent will track the progress of tasks and provide regular updates to the user.
4.  **Facilitate communication:** The agent can help schedule meetings, send reminders, and facilitate discussions among team members.
5.  **Generate project reports:** The agent can generate reports on project status, including burndown charts and velocity reports.

## Usage

To use this skill, provide the agent with the project details, including the project name, team members, and a high-level description of the project goals.

**Example:**

```
Manage the development of a new mobile app, "TaskMaster."

- Team members: Alice, Bob, Charlie
- Project goals: Develop a cross-platform mobile app for task management with a focus on simplicity and ease of use.
```
