## Workflow

This skill guides the agent in writing a well-structured and engaging blog post. The process is divided into the following steps:

1.  **Topic and Keyword Research:** Understand the blog post's topic and identify relevant keywords to target for SEO.
2.  **Outline Creation:** Structure the blog post with a clear introduction, body paragraphs, and a conclusion.
3.  **Content Generation:** Write the full content of the blog post, adhering to the outline and incorporating the identified keywords.
4.  **Review and Refinement:** Proofread and edit the blog post for clarity, grammar, and style.

## Usage

To use this skill, the agent should follow the workflow steps. The agent can use other tools for research and content generation.

### Example

**User Request:** "Write a blog post about the benefits of remote work."

**Agent's Execution:**

1.  **Topic and Keyword Research:** The agent uses the `search` tool to find articles and identify keywords like "remote work benefits," "work from home," and "distributed teams."
2.  **Outline Creation:** The agent creates an outline:
    *   Introduction: Hook the reader and introduce the topic.
    *   Body Paragraph 1: Increased flexibility and work-life balance.
    *   Body Paragraph 2: Reduced commute time and costs.
    *   Body Paragraph 3: Larger talent pool for employers.
    *   Conclusion: Summarize the benefits and offer a concluding thought.
3.  **Content Generation:** The agent writes the blog post, expanding on each point in the outline.
4.  **Review and Refinement:** The agent reviews the generated text for any errors and ensures it flows well.
