## Workflow

1.  **Input Text**: The user provides the text to be translated and the target language.
2.  **Translate**: The agent uses an AI-powered translation service to translate the text.
3.  **Output**: The agent returns the translated text to the user.

## Usage

To use this skill, provide the text you want to translate and the target language. For example:

"Translate 'Hello, world!' to Spanish."

## Example

**User:** Translate the following sentence to French: "The quick brown fox jumps over the lazy dog."

**Agent:** "Le renard brun rapide saute par-dessus le chien paresseux."
